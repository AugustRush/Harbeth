### CocoaPods

- If you want to import [**Metal**](https://github.com/yangKJ/ATMetalBand) module, you need in your Podfile: 

```
pod 'ATMetalBand'
```

- If you want to import [**OpenCV**](https://github.com/yangKJ/OpencvQueen) image module, you need in your Podfile: 

```
pod 'OpencvQueen'
```
